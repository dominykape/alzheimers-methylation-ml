# Alzheimer's Disease Prediction via DNA Methylation Profiling

Investigating the impact of 5-methylcytosine (5mC) and 5-hydroxymethylcytosine (5hmC) 
on Alzheimer's disease pathogenesis and disease prediction modelling.

## Overview
This project applies machine learning to clinical epigenomic data to identify DNA 
methylation biomarkers associated with Alzheimer's disease and build predictive models 
for disease classification.

## Dataset
- 162 clinical samples
- ~400,000 CpG methylation sites (C, 5mC and 5hmC)
- Source: Fox Chase Cancer Center (FCCC), Philadelphia, PA

## Methods
- Data cleaning and preprocessing
- Feature selection on high-dimensional epigenomic data
- Model building and comparison:
  - Logistic Regression
  - Linear Regression
  - Artificial Neural Network (ANN)
- Visualization of methylation patterns and model performance
